**Tech**

## 3️⃣ Moment – E-Commerce Platform

Production-style commerce backend with **order processing and payment integration**, deployed in a cloud environment.

**Features**

• Secure checkout using Razorpay
• Coupon and discount management
• Admin dashboard for product and order management

**Tech**

Node.js • Express.js • MongoDB • AWS EC2 • Nginx

---

## 4️⃣ AadhaarLens – OCR Processing API

Scalable backend API for extracting **structured data from Aadhaar card images using OCR**.

**Features**

• Secure authentication using JWT
• Image upload and OCR processing using Tesseract.js
• Persistent scan history and data storage
• Built using **Clean Architecture and Repository Pattern**

**Tech**

Node.js • TypeScript • Express.js • MongoDB

---
